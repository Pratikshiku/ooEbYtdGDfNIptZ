## Class Memoir System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/73d28206b5904461b03e67ba5dd73955/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Users, Class Monitors, Administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Class Information
5. News Information
6. Class Photo Album
7. Activity Information
8. Donation Information
9. Forum
10. Personal Center
11. My Posts
12. My Favorites
13. Change Password
14. Joined Classes
15. News Information
16. Class Photo Album Management
17. Activity Information Management
18. Donation Information Management
19. My Favorites Management




**Class Monitor Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Center
5. Class Information Management
6. Joined Classes Management
7. News Information Management
8. Class Photo Album Management
9. Activity Information Management
10. Donation Information Management



**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Class Monitor Management
5. Class Information Management
6. Class Joining Management
7. News Information Management
8. Class Photo Album Management
9. Activity Information Management
10. Donation Information Management
11. Forum Information Management
12. Banner Management
